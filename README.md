# 🌐 Text Translator

A React.js and Tailwind CSS application that translates English text into multiple languages using RapidAPI. The application provides a simple and responsive interface for real-time text translation, allowing users to communicate across different languages effortlessly.

## 🚀 Features

- Translate English text into multiple languages
- Language selection through a dropdown menu
- Real-time translation results
- Clean and responsive user interface
- RapidAPI integration for translation services
- Fast and user-friendly experience
- Built using modern React concepts

## 🛠️ Technologies Used

- React.js
- Tailwind CSS
- JavaScript (ES6+)
- RapidAPI
- Vite

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/ridd-coder/text-translator.git
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open your browser and visit:

```text
http://localhost:5173
```
